DUST COLLECTOR SYSTEM VISUALIZATION PACKAGE
===========================================

Quick Start:
1. Extract all files to the same folder
2. Run VIEW_DIAGRAMS.sh (Linux/Mac) or VIEW_DIAGRAMS.bat (Windows)
   OR manually run: python3 -m http.server 8000
   Then open: http://localhost:8000/diagrams_viewer_simple.html
3. OR just open the PNG files directly in any image viewer!

What's Included:
- 7 beautiful system diagrams (PNG format)
- Interactive HTML viewer
- Auto-generation scripts (Python)
- Complete documentation

For full instructions, see QUICK_START.md

Recommended for shop display: system_diagram_poster.png (300 DPI)
Recommended for interactive viewing: diagrams_viewer_simple.html

Questions? See VISUALIZATION_GUIDE.md for complete details.
